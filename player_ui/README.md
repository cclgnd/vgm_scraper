# Player UI Salvage Area

This folder is the clean target for the reimagined UI. It is intentionally not a
merge of the old `UI/` project.

Current rule:

- Use `UI/UI LAYER ONLY` as visual/reference material.
- Use `audio_engine/` as the playback implementation later.
- Use the scraper backend/API as the catalog and availability source later.
- Do not copy the old UI audio engine, downloader, scraper, or database layer.

The layout and button-placement contract lives in `docs/ui_reimagining.md`.
